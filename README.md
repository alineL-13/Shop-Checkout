# Device Shop Checkout

A simple and responsive Shop Checkout page built as a coding challenge from **devchallenges.io**, following the provided design specifications.

## Overview

This project consists of a shop checkout that includes

- Layered items
- Shopping items resume
- Coupon code area
- Contact information

The main goal of this challenge was to practice layout structure, semantic HTML, and responsive design using pure CSS.

## Built With

- HTML5
- CSS3

## Features

- Responsive design
- Semantic HTML structure
- Simple and clean UI

## Challenge Source

This project is based on a coding challenge available on **devchallenges.io**.
